IMS DB - Учебник

IMS - это системы управления информацией. IMS была разработана компанией IBM с Rockwell и компании Caterpillar в год 1966 год для Apollo программы отправить человека на Луну. Он начал системы управления базами данных революции и по-прежнему продолжает развиваться, чтобы удовлетворить требования к обработке данных.

IMS обеспечивает простое, надежное и стандартных условий для исполнения высокой производительности операций. IMS база данных используется в высоком уровне языков программирования COBOL для хранения данных в иерархической структуры и доступ к нему.

Целевая аудитория

Этот учебник предназначен для разработчиков программного обеспечения, заинтересованных в понимании концепции IMS база данных с нуля. Данное руководство содержит достаточного понимания данных МСМ management, где вы можете принять самостоятельно на более высокий уровень знаний.

Необходимые условия

Перед выполнением данного учебного курса, вы должны иметь базовые знания программирования COBOL. Базовые знания по базе данных концепций помогут вам понять IMS системы управления базами данных.
