Q 1 - In Plant Maintenance, arrange the following tasks in correct order?

  • Notification

  • Newly Created Work Order

  • Creation of Purchase request for work orders

  • Purchase Order created and parts Received

  • Parts issues to Work Order

  • Approval and Planning of Work Order

A - 6, 5,4,3,2,1

B - 3,1,2,4,5,6

C - 3,2,1,6,5,4

D - 1,2,3,4,5,6

E - None of These

Answer : D

Explanation

Key Steps that are included as part of Work Order Notification under SAP PM −

  • Notification
  • Newly Created Work Order
  • Creation of Purchase request for work orders
  • Purchase Order created and parts Received
  • Parts issues to Work Order
  • Approval and Planning of Work Order
  • Scheduling and Releasing Work Order
  • Work Order Execution
  • Closing a Work Order

Q 2 - Which of the following is incorrect about Preventive Maintenance?

A - Preventive Maintenance functionality is integrated with Controlling (CO) for cost allocations.

B - Single Cycle Plans have no maintenance strategy.

C - Preventive Maintenance is used when there is a malfunction occurs for any of the equipment.

D - Maintenance planning is used to generate the corresponding maintenance orders for all maintenance items affected at the due date.

Answer : C

Explanation

In SAP PM, when there is a malfunction occurs for any of the equipment which requires a response from Maintenance team. The process of correcting this situation is known as Breakdown maintenance.

Q 3 - What is the order type to perform Preventive Maintenance in Plant Maintenance?

A - PM01

B - PM02

C - PM03

D - PM04

Answer : C

Explanation

In SAP system, PM01 is the order type to perform Corrective maintenance and PM03 is for performing Preventive maintenance.

Q 4 - Which of the following maintenance is suitable to plan annual maintenance of a car or service of a Printer after 1 million copies?

A - Single Cycle Plan

B - Strategy Plan

C - Multiple Counter plan

D - Maintenance plan for service procurement

Answer : A

Explanation

Single cycle maintenance plan is used to define exactly one time-based or performance-based maintenance cycle, in which you specify the interval at which the maintenance plan should be executed.

It is one of the simplest maintenance plan used for Maintenance planning.

Example

A single cycle maintenance plan- annual maintenance of a car or service of a Printer after 1 million copies.
