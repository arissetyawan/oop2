SAP PM Mock Test I

Q 1 - Which of the following is not a function of SAP Plant Maintenance?

A - Inspection and Notifications

B - Corrective and Preventive Maintenance

C - Breakdown Maintenance

D - Repair

E - Material Requirement Planning (MRP) run

Answer : E

Explanation

Key functions as part of Plant Maintenance −

  • Inspection and Notifications
  • Corrective and Preventive Maintenance
  • Breakdown Maintenance
  • Repair
  • Other measures - charactertics and Key Figures

Q 2 - Which of the following submodule of Plant Maintenance is used to maintain high availability of the technical system?

A - Inspection

B - Repair

C - Preventive Maintenance

D - Breakdown Maintenance

Answer : C

Explanation

Inspection

Inspection is done to check the actual condition of a technical system.

Preventive Maintenance

Preventive maintenance is used to maintain high availability of the technical system. It includes maintenance planning and work scheduling activities for technical objects.

Repair

Repair involves all measures that can be performed to restore the ideal condition.

Q 3 - In Plant Maintenance, arrange the following tasks in correct order?

  • Notification

  • Newly Created Work Order

  • Creation of Purchase request for work orders

  • Purchase Order created and parts Received

  • Parts issues to Work Order

  • Approval and Planning of Work Order

A - 6, 5,4,3,2,1

B - 3,1,2,4,5,6

C - 3,2,1,6,5,4

D - 1,2,3,4,5,6

E - None of These

Answer : D

Explanation

Key Steps that are included as part of Work Order Notification under SAP PM −

  • Notification
  • Newly Created Work Order
  • Creation of Purchase request for work orders
  • Purchase Order created and parts Received
  • Parts issues to Work Order
  • Approval and Planning of Work Order
  • Scheduling and Releasing Work Order
  • Work Order Execution
  • Closing a Work Order

Q 4 - Which of the following statement is incorrect about SAP Plant Maintenance?

A - In Repair process, you can create required Purchase requisition, processed work orders to reduce the downtime.

B - Inspection is done to check the actual condition of a technical system.

C - In decentralize maintenance planning, company consists of multiple plants and there is only one plant for all the technical objects which is maintenance planning plant.

D - Technical objects are used to define the machine types that exists in an organization.

Answer : C

Explanation

Decentralize Maintenance Planning

In this scenario, company consists of multiple plants and each plant acts as its own maintenance planning plant and in SAP system all plants are mentioned as maintenance planning plant.

Q 5 - Which of the following activities is/are performed in Maintenance Plant?

A - Defining Task list as per maintenance plan

B - As per BOM in task list, you perform material planning.

C - Management and scheduling of maintenance plans

D - Maintenance Orders creation and execution

E - All of the above

Answer : E

Explanation

Maintenance plant for a technical object is known as plant in which you perform the maintenance tasks for the objects and planning is done. You can perform the following activities in Maintenance Planning Plant −

  • Defining Task list as per maintenance plan
  • As per BOM in task list, you perform material planning.
  • Management and scheduling of maintenance plans
  • Maintenance Orders creation and execution

Q 6 - When you classify a Functional Location, which of the following is not available under the class status?

A - Released

B - Locked

C - Installed

D - Incomplete

Answer : C

Q 7 - Which of the following is incorrect about Preventive Maintenance?

A - Preventive Maintenance functionality is integrated with Controlling (CO) for cost allocations.

B - Single Cycle Plans have no maintenance strategy.

C - Preventive Maintenance is used when there is a malfunction occurs for any of the equipment.

D - Maintenance planning is used to generate the corresponding maintenance orders for all maintenance items affected at the due date.

Answer : C

Explanation

In SAP PM, when there is a malfunction occurs for any of the equipment which requires a response from Maintenance team. The process of correcting this situation is known as Breakdown maintenance.

Q 8 - In Preventive Maintenance, a multiple counter plan do not have their own maintenance strategy and have the cycle sets. They contain a combination of time-based and performance-based cycles?

A - True

B - False

Answer : A

Q 9 - Which of the following is incorrect about Breakdown maintenance?

A - Equipment is not working properly.

B - Production is effected when a technical equipment went breakdown.

C - There is a malfunction occurs for any of the equipment.

D - You create PM03 order type.

E - None of these

Answer : D

Explanation

In Breakdown maintenance, you create a M2 malfunction notification and PM02 order type and equipment is repaired at the site.

Q 10 - Which of the following T-code is used to change the Notification?

A - IW21

B - IW22

C - IW23

D - IW24

Answer : B

Q 11 - Which of the following is incorrect about Maintenance strategy in Plant Maintenance?

A - Maintenance strategies are optional.

B - A maintenance strategy defines the rules for the sequence of planned maintenance work.

C - A maintenance strategy contains maintenance packages in which you can define the cycle in which the individual work should be performed.

D - Maintenance strategy is required to be defined with a multiple counter plan.

Answer : D
