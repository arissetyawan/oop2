Following quiz provides Multiple Choice Questions (MCQs) related to QC Framework. You will have to read all the given answers and click over the correct answer. If you are not sure about the answer then you can check the answer using Show Answer button. You can use Next Quiz button to check new set of questions in the quiz.

Q 1 - What does the pie chart in the Coverage Analysis window display?

A - Only the number of passed tests regarding the current cycle

B - The full test coverage for the test set, grouped according to test status

C - Only the number of passed tests regarding the current requirement

D - The full test coverage for the requirement, grouped according to test status

Answer : D

Explanation

The Coverage Analysis window displays the full test coverage for the requirement, grouped according to test status.

Q 2 - Which entities must be defined for Project Planning and Tracking?

A - Only Release and KPIs

B - Only Milestones, KPIs, and Scope Items

C - Release, Milestones, KPIs, and Requirements

D - Release, Milestones, KPIs, and Scope Items

Answer : D

Explanation

Release, Milestones, KPIs, and Scope Items must be defined for Project Planning and Tracking.

Q 3 - What happens if a team member tries to delete a representative requirement?

A - The deletion succeeds. The requirement coverage of the model entity is lost.

B - The deletion fails. The representative requirements are read-only and are therefore protected from deletion.

C - The deletion fails. The representative requirement is automatically made read-only, and an alert is optionally triggered.

D - The deletion succeeds. The representative requirement is moved to the Obsolete folder.

Answer : A

Explanation

The deletion succeeds. The requirement coverage of the model entity is lost.

Q 4 - What can be directly associated with defects?

  1. Requirements

  2. Test resources

  3. Test sets

  4. Test components

  5. Test instances

  6. Baselines

A - (1) (2) and (3)

B - (1) (3) and (5)

C - (2) (4) and (6)

D - (1) (2) and (5)

Answer : C

Explanation

Requirements, test components and test instances can be associated to defects.

Q 5 - Which parameter can be used to send an email message in plain text ?

A - MAIL SERMES

B - MAIL FORMAT

C - MAIL-MESSAGE-FORMAT

D - MAIL APPLETS

Answer : B

Explanation

MAIL FORMAT can be used to send an email message in plain text.

Q 6 - Your project manager requests that you send the execution status of a test set of 10 tests by tomorrow morning. There is a chance that the fifth test might fail. What should you do to ensure that the subsequent tests run?

A - Schedule the test run based on the Execution Condition.

B - Schedule the test run independently.

C - Schedule the test run by selecting the execution status Never Stop.

D - Schedule the test run based on the parameter.

Answer : B

Explanation

Schedule the test run independently to ensure that the subsequent tests run

Q 7 - A _________ is a view of a ALM window with the settings you apply to it.

A - Personal view

B - My view

C - Favorite view

D - User view

Answer : C

Explanation

Favorite view is a view of a ALM window with the settings you apply to it.

Q 8 - ALM reports can be generated from the _________

A - Requirements,Test Plan,Test Lab,Defects modules.

B - Requirements,Test Plan,Defects modules.

C - Requirements & Test Plan modules

D - Requirements,Test Plan,Test Lab modules.

Answer : A

Explanation

ALM reports can be generated from the Requirements,Test Plan,Test Lab,Defects modules.

Q 9 - You can also drag the test sets between lists.

A - True

B - False

Answer : A

Explanation

We can drag the test sets between lists.

Q 10 - A follow up flag is specific to your user login name. Anyone else viewing the record does not see your follow up alert.

A - True

B - False

Answer : A

Explanation

A follow up flag is specific to your user login name and anyone else viewing the record does not see your follow up alert.
