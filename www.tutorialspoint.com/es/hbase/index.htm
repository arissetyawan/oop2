    HBase - Tutorial

    Este tutorial proporciona una introducción a HBase, los procedimientos para configurar Hadoop HBase en sistemas de archivos, y maneras de interactuar con HBase shell. También describe cómo conectar a HBase utilizando java, y la forma de realizar las operaciones básicas en HBase usando java.

    Audiencia

    Este tutorial le ayudará a profesionales que aspiran a hacer una carrera en las Grandes Análisis de Datos usando Hadoop Marco. Los profesionales del Software, análisis profesionales y ETL los desarrolladores son los principales beneficiarios de este curso.

    Requisitos previos

    Antes de comenzar con este tutorial, vamos a asumir que usted ya conoce de Hadoop y arquitectura del Api, tener experiencia en escribir aplicaciones básicas utilizando java, y tener un conocimiento práctico de cualquier base de datos.
