Q 1 - SCM stands for?

A - Supply Continuity Management

B - Supply Crisis Management

C - Supplier Change Management

D - Supply Chain Management

Answer : D

Explanation

SAP Supply Chain Management is one of the key module in SAP ERP and controls Production Planning, business forecasting and demand planning, and helps the organization to manage their supply chain process in a dynamic environment.

Q 2 - Which of the following movement type is used for Goods Receipt in to Warehouse?

A - 101

B - 102

C - 103

D - 122

Answer : A

Q 3 - Quality is defined by the customer as?

A - an unrealistic definition of quality

B - a user based definition of quality

C - a manufacturing based definition of quality

D - a product based definition of quality

Answer : B

Q 4 - NETCH is used in?

A - Work Center

B - MRP Processing Key

C - BOM

D - Production Order

Answer : B

Explanation

NetChange in planning horizon −

This run considers the requirement in predefined planning horizon and not like net change which check the future requirements.

In this run, system consider only those material which has undergone some changes from their last MRP run.

Q 5 - In SAP SCM, which of the following master data is used to define sequence of activities performed at work center?

A - BOM

B - Material Master

C - Work Center

D - Routing

Answer : D

Explanation

Routing defines sequence of activities performed at work center. Routing plays in important role in calculating production cost, machine time and labor time.

Q 6 - Which of the following represents the active model and the active planning version?

A - model 000 and planning version 000

B - model 111 and planning version 111

C - model 100 and planning version 100

D - None of these

Answer : A

Explanation

Active model and the active planning version- model 000 and planning version 000 respectively.

Q 7 - In CIF integration model, to transfer data from R/3 system to SAP SCM what is PDS usage type?

A - PP/DS

B - PP/DS subcontracting

C - SNP & SNP subcontracting

D - None of the above

Answer : C

Explanation

In CIF integration model, you should select PDS type as SNP or SNP subcontracting. This allow R/3 data to be transferred to SAP SCM system and SNP PDS is generated from it in Supply Chain system.

Q 10 - Which of the following is used to fill the demand and supply gap in Production process?

A - BOM

B - MRP run

C - Work Center

D - Routing

Answer : B

Explanation

MRP run or planning run is an engine which is used to fill demand and supply gap. Issues & Receipts are called MRP Elements.
