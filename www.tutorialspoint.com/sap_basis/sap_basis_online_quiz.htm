Q 1 - Managing batch job scheduling and replication is part of?

A - System Administrator

B - Database Administrator

C - Transport and Batch job Administrator

D - DDIC Manager

Answer : C

Explanation

Transport and Batch job Administrator −

  • Managing batch job scheduling and replication
  • Manage SAP Control access in system landscape

Q 2 - In SAP system, you can create multiple clients on a single instance. It also provides data security as one user will client can’t see data of other user and also there is no need to install the software again?

A - True

B - False

Answer : A

Q 3 - Which of the following default client is known as SAP Reference client?

A - 000

B - 001

C - 066

D - 800

Answer : A

Explanation

000 This is called master client and is available when you install R/3 system.

Q 4 - Which of the following Transaction code is used to lock multiple users in SAP system?

A - SU01

B - SU10

C - SSC4

D - SCC5

Answer : B

Explanation

You can lock a user forcefully/automatically using transaction −

  • Transaction CodeSU01 for single user
  • Transaction Code SU10 for multiple users

Q 5 - Which of the following job class should be used for urgent or critical task?

A - Class A

B - Class B

C - Class C

D - None of the above

Answer : A

Explanation

Class A (High Priority)

This is used for urgent or critical task and must be scheduled with class A priority job. Class A job reserves one or more background work processes.

Class B (Medium Priority)

These jobs are executed after the completion of Class A high priority jobs.

Class C (Low Priority)

These job runs once class A and class B jobs are completed.

Q 6 - When a job is in Active state it can’t be unscheduled till it’s completed?

A - True

B - False

Answer : A

Q 7 - Which of the following can be used to delete multiple jobs in SAP system?

A - RSRTC

B - RSBTCDEL2

C - RSBTLDEC2

D - None of these

Answer : B

Explanation

To delete multiple jobs, use Transaction SE38. Use report RSBTCDEL2 (New version of RSBTCDEL). Old job logs will be deleted and will not show in the job overview.

Q 8 - In SAP system, all patches start with?

A - SPAK followed by Patch Type

B - SKAP followed by Patch Type

C - SAPK followed by Patch Type

D - PASK followed by Patch Type

Answer : C

Explanation

All patches start with SAPK followed by patch type.

  • ABAP (SAPKA620XXXX)
  • BASIS (SAPKB620XXXX)

Q 9 - Which of the following level, data from separate client is kept separate from each other?

A - Operating System Level

B - Database Level

C - Kernel Level

D - A and B

Answer : C

Q 10 - What is the Trusted RFC connection type to specify connection between ABAP systems?

A - Type I

B - Type T

C - Type 3

D - None of these

Answer : C

Explanation

Type 3:

This connection type is used to specify connection between ABAP systems. This is applicable for both type of RFCs, between ABAP systems and external calls to ABAP systems.
