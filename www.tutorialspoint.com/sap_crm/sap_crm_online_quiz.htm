Q 1 - Which of the following is a submodule in SAP Customer Relationship Management CRM?

A - SAP CRM Sales

B - SAM CRM Marketing

C - SAP CRM Analytics

D - SAP CRM Service

E - All of the above

Answer : E

Explanation

SAP CRM includes following sub modules that support key functions of Customer Relationship Management −

  • SAP CRM Sales
  • SAM CRM Marketing
  • SAP CRM Analytics
  • SAP CRM Service
  • SAP CRM Web Channel (E Marketing, CRM Mobile, etc.)
  • SAP CRM Interaction Center IC
  • SAP Hybris

Q 2 - In SAP CRM Web Client, which of the following area involves where changes happen with user actions and where user works?

A - Navigation bar

B - Header Area

C - Work Area

D - All of the above

Answer : C

Explanation

The entire frame in UI WebClient is known as Navigation bar. You can navigate to different pages in Navigation bar.

Header area is at the top and contains system links, work area title, saved searches, history, etc.

Work area is place where changes happen with user actions and where user works. Work area contains work center groups, overview pages, email inbox, etc.

Q 3 - In SAP CRM, which of the following areas are focused using CRM business by design?

A - Internet Sales

B - Partner Channel Management

C - Online and E-commerce sales

D - Customer Contact Management

Answer : D

Q 4 - Which of the following feature in CRM includes email or a telephone call or a customer visit or a follow up call, request to set the status to complete etc.?

A - Account Planning

B - CRM Sales

C - Activity Management

D - None of these

Answer : C

Q 5 - An Opportunity is defined as sales prospect, requested service or product, sales volume and a sales probability and is also known as Lead in CRM?

A - True

B - False

Answer : B

Explanation

A Lead is different from an opportunity in a manner that it is predecessor of an opportunity.

Q 6 - When a Quotation has been sent, which of the following can be used to take the first feedback from the customer?

A - Lead

B - RFx

C - Activity

D - Opportunity

Answer : C

Explanation

You can monitor the status of quotation by creating an activity. You can enter an activity for a specific time period (Performed in SAP CRM).

Q 7 - In SAP CRM, Availability check can be performed in SAP APO, SAP ECC or in external non-SAP system?

A - True

B - False

Answer : A

Q 8 - To perform campaign execution, you should have atleast one campaign created in CRM?

A - True

B - False

Answer : A

Q 9 - Which of the following can be used by customer to return a service part for which was not used?

A - Service Request

B - Complaint

C - Return

D - Service Order

Answer : C

Explanation

Return can be used for the goods delivered to a customer. A customer can also return a service part for which was not used.
