Q 1 - Which statement(s) is true about Multiproviders?

A - Virtual Infoprovider in SAP BW stores the data.

B - Multiprovider consists of InfoCubes, info objects and info sets

C - More than one info provider is required to build a Multiprovider

D - It is similar to joining the data tables

Answer : B

Explanation

To build MultiProvider, you have to use InfoCubes, info objects and info sets

Q 2 - What do you understand by data integrity in database?

A - Eliminating incorrect data

B - Eliminating duplicate data

C - Eliminating errors in data transformation

D - All of the above

Answer : A

Explanation

Data integrity deals with removing duplicate data in database.

Q 3 - DB connect relies on ABAP stack and Universal Data connector is based on Java stack?

A - True

B - False

Answer : A

Explanation

Both UD connect and DB connect are used for connection to any database system which is able to provide data for BI system. DB Connect relies on ABAP Stack of WAS and UD Connect is built on JAVA Stack of WAS.

Q 4 - Which of the following can be used for reporting in SAP BW?

A - BEx Analyzer

B - Lumira

C - BICS

D - BEx query

Answer : A

Explanation

BEx Analyzer is one of reporting environment in SAP BW and is integrated with MS Excel.

Q 5 - In SAP BW, InfoObjects in dimension tables are called?

A - Key Figures

B - Characteristics

C - Delta Load

D - Attributes

Answer : B

Explanation

InfoObjects in Dimension tables are called characteristics and InfoObject in fact tables are called key figures.

Q 6 - In SAP BW, to create new InfoObjects to store data, which of the following tab is used in Administration workbench?

A - Administration

B - Data Source

C - Modeling

D - Translation

Answer : C

Explanation

In RSA1, you use Modeling tab to create new objects to store BI data.

Q 7 - Which of the following is not a view type while managing process chains in BW system?

A - Planning View

B - Checking View

C - Log View

D - Status View

Answer : D

Explanation

There are three main views in icon bar while managing process chains −

  • Planning View
  • Checking View
  • Log View

Q 8 - A company wants to set up BI reporting on Sales data from source system without extraction. Which of the following SAP BW feature support this?

A - MultiProvider

B - Delta Load

C - Virtual Provider

D - InfoSets

Answer : C

Explanation

You can access BI data from source system without extraction using Virtual Providers.

Q 9 - Which of the following is component of Business Explorer suite?

A - Web Analyzer

B - Report Designer

C - BICS

D - BEx Query Designer

E - PSA

Answer : E

Explanation

BEx suite is reporting environment in SAP BW and it doesn't contain Persistent Staging Area of SAP BW

Q 10 - Which of the following class contains majority of authorization objects in BI?

A - AS

B - DS

C - RS

D - ZS

Answer : C

Explanation

Authorization objects are grouped according to authorization object classes and majority of objects are in class RS in BI.
